# Goroutines to Fibers: A Rails Developer's Survival Guide

RubyJam 2026 lightning talk slides by Mu-Fan Teng ([@ryudoawaru](https://github.com/ryudoawaru)).

**Live:** https://rubyjam2604.ryudo.tw

## Development

```bash
yarn install
yarn dev      # http://localhost:3030
yarn build    # output to dist/
yarn export   # export to PDF
```
